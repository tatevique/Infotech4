Info Tech – Open Source

This section highlights an open-source tool used in libraries for research and usability testing: GazeParser.

About GazeParser

GazeParser is a Python-based software library designed to capture, analyze, and visualize eye-tracking data. Originally developed for psychological research, it is increasingly used in human-computer interaction (HCI) studies, including applications within library environments. Libraries can use GazeParser to better understand how patrons interact with digital catalogs, self-checkout machines, and informational kiosks.

Features of GazeParser:
  • Supports low-cost eye trackers
  • Real-time data recording and processing
  • Customizable experiment creation
  • Exports data for further analysis in Python or Excel

By leveraging eye-tracking data, librarians and system designers can optimize user interfaces for clarity, accessibility, and efficiency.
